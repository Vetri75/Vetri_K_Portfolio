Secure File Storage on Cloud Using Hybrid Cryptography

*This is college team project, Contributed to the Java backend development of a secure file storage system using hybrid cryptography techniques.Developed RESTful APIs for secure file uploads and access control.

Overview:

Developed a secure file storage system using a hybrid cryptography model combining Elliptic Curve Cryptography (ECC) and AES encryption to enhance data security in cloud environments. The system ensures confidentiality, integrity, and controlled access by encrypting data before uploading and decrypting it only for authorized users.

Tech Stack:

Backend-Java, NetBeans IDE

Cryptography:ECC, AES, ECDH Algorithms

Database:Relational Database Server (AWS RDS)

Key Features:

  • Hybrid Encryption - Used ECC for key exchange and AES for data encryption to strengthen security.
  • Authentication Mechanism - Implemented a user authentication system with unique ID generation & OTP-based verification to prevent unauthorized access.
  • Secure Key Exchange - Integrated Elliptic Curve Diffie-Hellman (ECDH) for secure key transmission.
  • Role-Based Access - Ensured only authorized users could decrypt and access files.
  • Cloud Storage Integration - Stored encrypted files securely on the cloud to enable remote access while maintaining security.
